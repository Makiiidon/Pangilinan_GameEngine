Instructions on how to run the code:
1. Open the folder found in the root folder PangilinanGameEngine.
2. Open PangilinanGameEngine.sln file to open the project.
3. To local the main.cpp file go to PangilinanGameEngine -> App -> main.cpp.
4. Make sure that the solution configuration is set to debug.
5. Make sure that the Solution Platform is x86.
6. Click ctrl + f5 to start the program.
